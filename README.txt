Реализация алгоритма сортировки QSort Данный проект представляет собой реализацию алгоритма Быстрой сортировки. Проект состоит из трёх частей:

Проект, в котором находится файл с программой (ConsoleApplication2)
Файл, где продемонстрировано тестирование(ConsoleApplication3)
Функция QSort работает с типами данных типа int. Внутри функции используется команда swap, для смены местами элементов массива. Все элементы массива задаются случайно, на выбранном промежутке, количество элементов статично и задано пользователем. При тестировании массив вводится вручную и проверяется соответствие сортировке.

Проект распространяется по лицензии GNU GPL, и может свободно использоваться в любых целях.
